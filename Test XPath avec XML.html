Exemple de test XPath avec XML

Voici le fichier XML utilisé pour tester XPath :

<etudiants>
    <etudiant>
        <nom>Ahmed Ben Salem</nom>
        <moyenne>15.46</moyenne>
        <niveau>1</niveau>
    </etudiant>
    <etudiant>
        <nom>Maha Ben Salah</nom>
        <moyenne>10.25</moyenne>
        <niveau>1</niveau>
    </etudiant>
    <etudiant>
        <nom>Ali Mahjoub</nom>
        <moyenne>12.86</moyenne>
        <niveau>1</niveau>
    </etudiant>
</etudiants>

Voici des requêtes XPath que tu peux tester :

//etudiant/nom
//etudiant[moyenne > 14]/nom
count(//etudiant)
